# Use Cases

## Overview

This document set describes the interactions between actors and the Metadata-Driven Secure Plugin Runtime.

Use Cases bridge the gap between requirements and implementation by describing how users, administrators, plugins and external systems interact with the Runtime.

Each Use Case references the related:

- Functional Requirements (FR)
- Business Rules (BR)
- Non-Functional Requirements (NFR)
